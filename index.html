Free Your Boredom API

Something to do:

The API I chose genrates random things to do every time the web page is refreshed.

I found this API on https://mixedanalytics.com/blog/list-actually-free-open-no-auth-needed-apis/

Not only does it generate random activities to do in the JSON data return, but it also tells the user what type of activity it is suggesting you, how many participants it requires, and the price it may cost to complete this activity. For simplicity purposes, I only used the activity section of the JSON data.

This API could be used when internet service is out, just like when google shows the running dinosaur game when there is no internet connection. This API could also be used on any website that could use these suggestions as a slide show.
